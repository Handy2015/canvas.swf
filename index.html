FlashCanvas

An implementation of the HTML 5 Canvas API in Flash for Internet Explorer.

Usage

Copy the files flashcanvas.js and flashcanvas.swf in the same folder somewhere on your server. Include the file flashcanvas.js in your source as follows:

<!--[if IE]><script type="text/javascript"
     src="/path/to/flashcanvas.js"></script><![endif]-->

That's it! All <canvas> elements in markup or generated using document.createElement('canvas') will be interactive using the Canvas API, allowing you to make seamless, cross-browser graphics using the canvas element.

How to get involved

FlashCanvas is a work in progress. You can help in any of the following ways: implementing more parts of the Canvas API, helping report any issues or bugs, or using FlashCanvas in your own projects. For more information, contact Tim Ryan.

FlashCanvas is authored using the haXe programming language and JavaScript.
