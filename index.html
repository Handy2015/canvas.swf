FlashCanvas

An implementation of the HTML 5 Canvas API in Flash for Internet Explorer.

Usage

Copy the files flashcanvas.js and flashcanvas.swf in the same folder somewhere on your server. Include the file flashcanvas.js in your source as follows:

<!--[if IE]><script type="text/javascript"
     src="/FlashCanvas/bin/flashcanvas.js"></script><![endif]-->

That's it! All <canvas> elements in markup or generated using document.createElement('canvas') will automatically be bound to the Canvas API, allowing you to make seamless, cross-browser code using the canvas element.

Caveats

  • Canvas elements in markup will not be bound until the onreadystatechange event for the document fires for COMPLETE.
  • Canvas elements generated using innerHTML or other raw HTML-writing methods will not be bound.
